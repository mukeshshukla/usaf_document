Class Image_Graph_Axis_Radar

Displays an 'X'-axis in a radar plot chart.

This axis maps the number of elements in the dataset to a angle (from 0- 360 degrees). Displaying the axis consist of drawing a number of lines from center to the edge of the 'circle' than encloses the radar plot. The labels are drawn on the 'ends' of these radial lines.
