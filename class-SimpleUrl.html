Class SimpleUrl

URL parser to replace parse_url() PHP function which got broken in PHP 4.3.0. Adds some browser specific functionality such as expandomatics. Guesses a bit trying to separate the host from the path and tries to keep a raw, possibly unparsable, request string as long as possible.
